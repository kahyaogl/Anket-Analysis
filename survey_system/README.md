# ACEY - Yapay Zeka Destekli Anket Analiz Paneli

ACEY, ham anket verilerini MS SQL Server'dan alan, BERT ve Gemini API kullanarak duygu ve konu analizi yapan, sonuçları görselleştirip PDF raporu sunan bir sistemdir.

## Özellikler
-  **Duygu Analizi:** BERT modeli ile Türkçe duygu tespiti.
-  **Konu Başlıklandırma:** LLM (Gemini) desteğiyle akıllı kategorizasyon.
-  **PDF Raporlama:** Grafiklerle zenginleştirilmiş analiz çıktıları.
-  **Web Arayüzü:** Streamlit tabanlı interaktif dashboard.
